# SQL Project: Insurance Premium Payment Analysis

## Overview

This project analyzes insurance premium payment data using SQL.
The goal is to identify failed payments, analyze premium collection trends, and evaluate employee processing performance.

## Dataset Tables

The project contains three main tables:

* **loans** – Information about customer loans
* **premium_payments** – Records of premium payments
* **employees** – Employees who processed the payments

## SQL Concepts Used

* SELECT
* WHERE
* GROUP BY
* ORDER BY
* JOIN
* CASE statements
* Aggregate functions

## Key Analysis Performed

1. Total premium collected
2. Failed payment detection
3. Premium collected per loan
4. Employee payment processing performance
5. Payment status alerts using CASE

## Key Insights

* Identified failed premium payments
* Analyzed loan-wise premium collection
* Evaluated employee transaction processing efficiency
